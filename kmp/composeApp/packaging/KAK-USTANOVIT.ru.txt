3X-UI Manager — менеджер панели 3x-ui для macOS
================================================

ТРЕБОВАНИЯ
----------
• Mac на Apple Silicon (M1/M2/M3/M4).
  На процессорах Intel приложение не запустится.
• Java НЕ нужна — она уже встроена в приложение.


УСТАНОВКА
---------
1. Перетащите «3X-UI Manager» в папку «Applications» (Программы)
   в этом же окне DMG.
2. DMG можно извлечь (значок ⏏ в Finder).


ПЕРВЫЙ ЗАПУСК (важно!)
---------------------
Приложение подписано «ad-hoc» и не нотаризовано в Apple, поэтому при
первом запуске macOS его заблокирует («повреждено» или «неизвестный
разработчик»). Это нормально — нужно один раз снять карантин.

Способ 1 — Терминал (самый надёжный):
  Откройте «Терминал» и выполните одну команду:

      xattr -dr com.apple.quarantine "/Applications/3X-UI Manager.app"

  После этого запускайте приложение как обычно (двойным кликом).

Способ 2 — через интерфейс:
  Правый клик по «3X-UI Manager» → «Открыть» → в диалоге ещё раз
  «Открыть».
  (Если macOS пишет именно «повреждено» — поможет только Способ 1.)


ЧТО ДАЛЬШЕ
----------
При первом запуске введите:
  • URL панели (например https://ваш-домен/путь)
  • API-токен (создаётся в панели: Settings → Security → API Token).
Нужна панель 3x-ui версии 3.3.0 или новее.
